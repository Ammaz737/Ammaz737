## 🔨 What I'm Building

### Crime Intelligence RAG Platform *(GCS Pvt Ltd — Production)*
An AI-powered platform for law enforcement agencies to extract actionable intelligence from large-scale FIR records. Built with LangGraph multi-agent architecture, advanced RAG pipelines with query rewriting, ChromaDB vector search, Redis caching, and Docker on-premise deployment.

### Social Media Listening Agent *(GCS Pvt Ltd — Production)*
Autonomous agents that continuously monitor sentiment across platforms, surface emerging discussion trends, and contextually respond to posts — zero manual intervention required.

### HireFlow — AI HR Automation *(GCS Pvt Ltd)*
End-to-end recruitment automation: semantic CV screening, multi-criteria candidate ranking, AI-generated interviews, and automated email workflows — built on LangChain + OpenAI APIs.
